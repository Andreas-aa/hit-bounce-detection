# Hit and Bounce Detection

## Description
This repository contains a supervised machine learning solution for detecting **Hit** and **Bounce** events.

The project provides:
- Two detection functions (Hit and Bounce)
- A trained supervised model
- All required files to run the solution

## Repository Structure
- `main.py`: Contains the Hit and Bounce detection methods
- `requirements.txt`: Python dependencies
- `model/`: Trained supervised model
- `data/`: Sample input data (if applicable)
